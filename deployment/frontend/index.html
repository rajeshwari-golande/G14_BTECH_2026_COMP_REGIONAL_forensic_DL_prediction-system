Forensic Fingerprint Analysis System

4th Year Computer Engineering Project — UI on Vercel, inference API on Railway

About the project

Project title

Forensic prediction of blood group and gender from fingerprints using deep learning

Project description

This project uses deep learning and computer vision on fingerprint images for forensic-style analysis: separate models estimate blood group and gender with confidence scores.

The live demo calls a deployed API so you get real predictions in the browser—no local Python required for viewers.

Key features

  • Image preprocessing — normalization and enhancement pipeline before inference
  • Dual-model architecture — dedicated models for blood group and gender
  • Confidence scores — outputs with per-class blood group breakdown
  • Three-tab UI — About (this page), Members, and Predict for demos
